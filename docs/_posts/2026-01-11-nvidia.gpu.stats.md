```bash
## default (1s, 100MB)
lsd-nvidia.gpu.stats-log

## 2s interval, default size
lsd-nvidia.gpu.stats-log 2

## 1s interval, 50MB rolling
lsd-nvidia.gpu.stats-log 1 --size=50

## env-based default
export LSD_NVIDIA_GPUSTATS_LOG_MB=500
lsd-nvidia.gpu.stats-log
```
